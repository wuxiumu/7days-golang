## 指针的概念    

指针是一种变量，它存储着另一个变量的地址。

## 指针的声明和初始化

```go
var ptr *int
```

## 指针的赋值

```go
var a int = 10
var ptr *int
ptr = &a
```

## 指针的使用

```go
*ptr = 20
```
